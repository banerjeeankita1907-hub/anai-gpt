# ANAI – On‑Device GPT

A **real transformer language model** running entirely in your browser. No API keys, no servers, no cloud. Built with Transformers.js and ONNX Runtime.


Created by **Ankita Banerjee**.

## How it works
- Loads `LaMini-Flan-T5-77M` from a public CDN (Hugging Face).
- Runs inference in WebAssembly using ONNX Runtime.
- Includes fallback rule‑based responses for instant answers.
